Liberation Sans Regular Subset Numeric Degrees
======

Numeric subset of Liberation Sans Regular, used for dynamic text performance (changing 12+ text values per frame was a major bottleneck when using SVG text).  (uses the SIL Open Font License, http://en.wikipedia.org/wiki/SIL_Open_Font_License)

Extracted paths and x-advances for the glyphs using FreeType.

Developed by Jonathan Olson for PhET Interactive Simulations using font developed by Steve Matteson of Ascender Corporation
